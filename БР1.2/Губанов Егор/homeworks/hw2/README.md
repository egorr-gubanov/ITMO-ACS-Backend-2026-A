# ДЗ2 — API аренды недвижимости

Спецификация на TypeSpec, сборка в OpenAPI 3.1, просмотр в Swagger UI.

```bash
npm install
npm run build
npm start
```

Порт **8001** (8000 занят примером LMS в `examle`). Откройте http://localhost:8001/docs

Сгенерированная схема: `tsp-output/schema/openapi.yaml`

Отчёт: `ДЗ2_Губанов Егор_БР1.2.md` — скопируйте разделы в [шаблон курса](https://docs.google.com/document/d/1aAUawxv6_5k_Na7bLqrfUFANodyl89uPHXY4IKXS8WE/edit?usp=sharing) и сохраните как PDF (`ДЗ2_Губанов Егор_БР1.2.pdf`). Либо откройте `.md` в редакторе и «Печать → PDF».
